## はじめに

営業リスト作成の外注は、多くの中小企業が「手っ取り早い」と感じるソリューションです。しかし、外注を続ければ続けるほど**コストは積み上がり、データは社外に流出**します。

一方、AI内製化は「初期投資が必要だが、長期的にはコストが逓減する」モデル。この記事では、両者のコスト構造を詳細に比較し、御社にとっての最適解を見つける判断基準を提示します。

## 外注のコスト構造

### リスト購入型（データベースサービス）

| 項目 | コスト |
|---|---|
| 1回のリスト購入（1,000件） | 5万〜15万円 |
| 月額サービス利用料 | 3万〜10万円 |
| カスタマイズオプション | 1万〜5万円/回 |
| **月額合計** | **8万〜30万円** |
| **年間合計** | **96万〜360万円** |

### リサーチ代行型（BPO）

| 項目 | コスト |
|---|---|
| リサーチャー人件費（1名） | 20万〜40万円/月 |
| PM・管理費 | 5万〜10万円/月 |
| ツール利用料 | 3万〜5万円/月 |
| **月額合計** | **28万〜55万円** |
| **年間合計** | **336万〜660万円** |

### 外注の隠れたコスト

上記の金額に加え、以下の「見えないコスト」があります：

- **コミュニケーションコスト**: 要件伝達、修正依頼に月5〜10時間
- **リスト精査コスト**: 納品リストを社内で精査する工数
- **機会損失**: リスト納品まで1〜2週間のタイムラグ
- **ノウハウの非蓄積**: データと経験が社外にとどまる

## AI内製化のコスト構造

| 項目 | コスト |
|---|---|
| 初期設計・構築費 | 50万〜150万円（一括） |
| 月額ツール利用料 | 3万〜10万円 |
| 月額運用サポート | 3万〜5万円 |
| **月額合計** | **6万〜15万円** |
| **年間合計（初年度）** | **122万〜330万円** |
| **年間合計（2年目以降）** | **72万〜180万円** |

### AI内製化の隠れたメリット

- **データが社内に蓄積**: 使い続けるほどAIの精度が向上（成約パターンの学習）
- **リアルタイム更新**: 情報の鮮度が常に最新
- **スケーラビリティ**: リスト1,000件でも10,000件でもコスト変わらず
- **担当者の成長**: 「リスト作成」から「リスト戦略立案」へ業務がレベルアップ

## 3年間のTCO（総保有コスト）比較

| 年 | 外注（購入型中程度） | AI内製化 |
|---|---|---|
| 1年目 | 200万円 | 200万円 |
| 2年目 | 200万円 | 120万円 |
| 3年目 | 200万円 | 120万円 |
| **3年間合計** | **600万円** | **440万円** |
| **累積差額** | — | **160万円の削減** |

さらにAI内製化の場合、2年目以降はAIの精度向上により**リストの質が上がり、成約率の向上**も期待できます。この成約率の向上分を売上に換算すると、実質的なROIはさらに大きくなります。

## どちらを選ぶべきか

### 外注が向いている場合

- **AI内製化の初期投資が難しい**場合
- 新規市場の開拓で**一時的に大量のリストが必要**な場合
- 自社に**過去の成約データがほとんどない**場合
- **事業の立ち上げ期**で、ターゲット自体が不確定な場合

### AI内製化が向いている場合

- 営業リスト作成が**継続的な業務**である場合
- CRMに**過去1年以上の成約データ**がある場合
- 月間のリスト作成コストが**10万円以上**の場合
- リストの質を**継続的に改善したい**場合
- **データの社内蓄積**を重視する場合

### ハイブリッドアプローチ

最も現実的な選択肢は、**最初は外注で始め、データが蓄積したらAI内製化に移行**するハイブリッドアプローチです。

1. 立ち上げ期（〜6ヶ月）：外注リストで営業活動を開始
2. 蓄積期（6ヶ月〜1年）：成約データを蓄積しながらAI導入を設計
3. 移行期（1年〜）：AI内製化を開始、段階的に外注を縮小
4. 運用期（1.5年〜）：完全AI内製化、外注はスポット利用のみ

## 専門家に任せるべきポイント

AI内製化の初期設計（成約パターン分析、スコアリングモデル構築、データソース連携）は専門家のサポートが効果的です。

## まとめ

営業リスト作成の外注とAI内製化は「どちらが正解か」ではなく、**自社のフェーズ（立ち上げ期 vs 成長期）と累積データ量で最適解が変わる**判断です。

長期的にはAI内製化の方がコスト・質ともに有利ですが、**データの蓄積がないうちは外注から始める**のが現実的です。
