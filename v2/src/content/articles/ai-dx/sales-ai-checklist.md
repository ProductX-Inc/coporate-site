## はじめに

![DX戦略チェック](/articles/ai-dx/images/dx-strategy-hero.webp)

「営業チームにAIを入れたいが、本当に今のタイミングか？」「うちの会社にAIは早すぎるのではないか？」——こうした疑問に対し、**客観的な判断基準**を提供します。

この記事は、営業チームへのAI導入を検討する際のチェックリストです。自社の準備状況を診断し、導入の「Go / Not Yet」を判断してください。

## セクション1：データの準備状況

AI活用の成否は**データの質と量**で80%決まると言われています。

| チェック項目 | YES | NO |
|---|---|---|
| CRMに過去1年以上のデータがあるか | +2点 | 0点 |
| 商談データ（金額・ステージ・結果）が記録されているか | +2点 | 0点 |
| 顧客の業種・規模・担当者情報が整備されているか | +1点 | 0点 |
| データの入力率は80%以上か | +2点 | 0点 |
| データの入力精度に自信があるか | +1点 | 0点 |

**小計: /8点**

- 6点以上：データの準備は十分。AI導入に適したタイミング
- 3〜5点：データ整備と並行してAI導入を検討
- 2点以下：まずデータ整備から始めるべき

## セクション2：業務の課題明確度

AIは**「課題」が明確な場合にのみ効果を発揮**します。漠然と「効率化したい」ではAIを活用しきれません。

| チェック項目 | YES | NO |
|---|---|---|
| 最も時間を使っている非売上業務を特定できるか | +2点 | 0点 |
| その業務に費やしている時間を定量化できるか | +1点 | 0点 |
| 改善した場合の売上インパクトを試算できるか | +2点 | 0点 |
| 営業チーム全体の課題として共有されているか | +1点 | 0点 |
| 経営者と現場の認識にギャップがないか | +2点 | 0点 |

**小計: /8点**

- 6点以上：課題が明確。AI導入の方向性が定まりやすい
- 3〜5点：課題の深掘りが必要だが、AI導入検討は可能
- 2点以下：まず業務分析と課題の言語化から

## セクション3：組織の変革準備度

AIは**組織が受け入れてこそ効果を発揮**します。

| チェック項目 | YES | NO |
|---|---|---|
| 経営者がAI導入に予算を確保する意思があるか | +2点 | 0点 |
| 営業マネージャーが新しいツールに前向きか | +2点 | 0点 |
| 営業メンバーの中にITリテラシーの高い人がいるか | +1点 | 0点 |
| 過去にツール導入（CRM等）を成功させた経験があるか | +1点 | 0点 |
| DXの推進担当者（兼任可）をアサインできるか | +2点 | 0点 |

**小計: /8点**

- 6点以上：組織の受け入れ体制は良好
- 3〜5点：一部の関係者の巻き込みが必要
- 2点以下：組織の意識改革が先

## セクション4：投資対効果の見込み

| チェック項目 | YES | NO |
|---|---|---|
| 月間のリスト作成に10時間以上使っているか | +1点 | 0点 |
| 月間の日報/レポート作成に8時間以上使っているか | +1点 | 0点 |
| 月間の提案書作成に15時間以上使っているか | +1点 | 0点 |
| 問い合わせへの初回対応に平均1時間以上かかっているか | +2点 | 0点 |
| 営業チームの成約率が業界平均を下回っているか | +2点 | 0点 |
| 年間の営業関連コスト（人件費含む）が3,000万円以上か | +1点 | 0点 |

**小計: /8点**

- 6点以上：AI投資のROIが高い。積極的に検討すべき
- 3〜5点：一部の領域でROIが期待できる
- 2点以下：現時点ではROIが限定的。優先度は高くない

## 総合判定

| 合計スコア | 判定 | 推奨アクション |
|---|---|---|
| 24〜32点 | **Go** 🟢 | 今すぐAI導入を開始。専門家との設計フェーズへ |
| 16〜23点 | **Conditional Go** 🟡 | スコアが低い領域を改善しつつ、小規模から開始 |
| 8〜15点 | **Not Yet** 🔴 | まずデータ整備と課題の明確化に注力 |
| 7点以下 | **準備フェーズ** ⚪ | CRM導入とデータ蓄積の仕組みづくりから |

## スコア別の推奨アクション

### 24〜32点（Go）

1. 課題の優先順位付け
2. 専門家との設計ワークショップ
3. パイロット導入（最も効果が高い領域から）
4. 3ヶ月後に効果検証 → 横展開

### 16〜23点（Conditional Go）

1. スコアが低い領域のギャップ分析
2. データ整備と課題の言語化を並行
3. 最もシンプルな領域（日報自動化等）から小さく始める
4. 成功体験を積みながら組織の準備度を向上

### 15点以下（Not Yet / 準備フェーズ）

1. CRMの導入・定着を最優先
2. 業務時間の計測と課題の可視化
3. 経営者と現場のDXに対する認識統一
4. 6ヶ月後にチェックリストを再実施

## 専門家に任せるべきポイント

チェックリストの結果を踏まえた**AI導入のロードマップ設計**、課題の優先順位付け、ツール選定は専門家の伴走が効果的です。
